# Agent Instructions

- Use Next.js App Router under `src/app` (do not use the Pages Router).
- Use TypeScript and keep ESLint clean.
- Put reusable logic in `src/lib` and UI components in `src/components`.
- Never commit secrets. Use `.env.local` for environment variables.
- Prefer Server Components and Server Actions; add `"use client"` only when necessary.
- Keep file names, exports, and folder structure consistent and minimal.
